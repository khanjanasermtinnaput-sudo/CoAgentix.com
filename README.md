# Co.AI — CoAgentix.com

> Many minds, one intelligence.

The premium marketing site for **Co.AI** — a multi-agent AI platform combining
intelligent routing, memory, and orchestration.

## Stack

- **Next.js 14** (App Router) + **TypeScript**
- **Tailwind CSS** — white/black monochrome theme, CoAgentix Orange (`#FF6A00`) accent
- **Framer Motion** — scroll reveals, parallax, counters, staggered motion
- **GSAP** — animation utilities
- **Lenis** — cinematic smooth scrolling
- **Inter** + **Geist** typography

## Homepage sections

1. Hero — animated background, floating shapes, mouse parallax, scroll motion
2. Technology Showcase — TMAP v2, RAA, Orchestrator, Memory, Multi-Agent, Voting, Titan
3. Interactive Architecture — animated pipeline diagram
4. TMAP Deep Dive — timeline storytelling
5. RAA System — animated retrieval workflow
6. Benchmarks — animated counters + dashboard
7. Product Ecosystem — bento grid
8. Company Vision — scroll text reveal
9. Final CTA

## Develop

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # production build
npm start        # serve production build
```

## Notes

- Fully static, accessibility-minded (skip link, reduced-motion support, focus states).
- SEO: Open Graph, Twitter cards, JSON-LD Organization schema.
